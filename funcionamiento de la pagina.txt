El navegador solicita mediante una peticion http la pagina web.
El servidor responde con la pagina web.
Luego de establecida la conexion con el servidor, un codigo JavaScript conecta la url del servidor a una de webSocket.
Despues se envia una solicitud del navegador para pasar la conexion de http a una de websocket.
Por ultimo se establece una conexion bidireccional para la comunicacion entre clientes
